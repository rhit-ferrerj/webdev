1.4 Reading Assignment

  • What does HTML consist of?

    HTML consists of a series of elements and attributes which are used to mark up all the components of a document to structure it in a meaningful way.

  • Describe the figure:

    The figure shows an HTML element which consists of the opening and closing tags as well as the content between the tags.

  • Can you nest elements?

    Yes, when an element has other elements in it like the root "html" element, it is considered as having other elements nested in it.

  • Provide examples of void elements.

    All self-closing tags represented by one tag. Examples include elements starting with tags like: br, col, hr, img, input etc.

  • Provide examples of replaced elements.

    Replaced elements are replaced by objects, e.g. a graphical UI. Examples include elements starting with the img or input tag.

  • Where do you place attributes?

    Extra bits of space seperated name-value-pairs only appear in the opening tag.

  • Name all attributes:

    src, alt, ismap
